# Push to GitHub - Quick Guide

## Step 1: Create GitHub Repository

1. Go to: https://github.com/new
2. Repository name: `vortex-shield`
3. Description: `Parametric microinsurance platform with AI-powered claims processing for gig workers`
4. **Keep it PUBLIC** (for hackathon visibility)
5. **Do NOT initialize** with README (we already have one)
6. Click "Create repository"

## Step 2: Run These Commands

After creating the repo, run:

```bash
cd C:\Users\rakti\CascadeProjects\vortex-shield

git remote add origin https://github.com/RaktimChandra/vortex-shield.git

git branch -M main

git push -u origin main
```
